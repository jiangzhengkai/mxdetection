## Introduction

mxdetection is an open source object detection toolbox based on Mxnet.

### Major features

- **Modular Design**
- **Efficient**
- **State of the art**

### Install Dependencies

- **mxnet 1.3.0**

- **python 2.7**

- **Cython**
```
pip install Cython
```
- **numpy**
```
pip install numpy
```
- **easydict**
```
pip install easydict
```
- **cv2**
```
pip install opencv-python
```
